Portfolio Personal

Uno de los proyectos realizados es mi Portfolio Personal creado en el curso de Full Stack Jr. brindado por Argentina Programa. Para llevar a cabo éste proyecto, en la parte de Front End se utilizó HTML, CSS, Bootstrap y Typescript a través del desarrollo en el framework Angular. En la parte de Back End se diseñó la base de datos en MySQL y se utilizó el lenguaje Java para el manejo de los mismos. El diseño de la API para conectar con el Front se realizó mediante Java Spring Boot. Finalmente, el deploy del Front se realizó mediante Firebase y el del Back a través de Heroku conectando la base de datos con Clever Cloud.

Dashboard para Análisis de Datos

Éste proyecto, realizado en el curso de Data Analytics en Codehouse, consistió en el desarrollo de un dashboard de control en Power BI, con el fin de analizar indicadores estratégicos y operativos. Como temática, se decidió analizar los datos de pacientes trasplantados en efectores públicos y privados, de la Ciudad de Buenos Aires, desde el año 2012 al año 2020, con datos provenientes de bases de datos del gobierno de la Ciudad. El objetivo del análisis es determinar cuáles son las principales características de las personas que fueron trasplantadas en el período del año 2012 al año 2020 (grupos etarios, promedio de edad, género), cuáles fueron los tipos y motivos del trasplante (órganos implantados) y cuál será la evolución de los pacientes trasplantados, con el fin de prevenir los factores determinantes que ocasionan dichos trasplantes.
